# Codex Personal Skills

Personal Codex skills that are safe to sync across devices.

## Included

- `browser-screenshot-delivery`
- `codex-windows-oauth-repair`
- `windows-microsoft-store-repair`

## Excluded

These were intentionally left out:

- `autodl-devbox`
- `payment-qr-handoff`
- `pku-course`
- `pku-mail`
- `quark-drive`

## Install On Another Device

Clone this repository, then run one of the installers from the repository root.

Windows PowerShell:

```powershell
.\install.ps1
```

macOS or Linux:

```bash
./install.sh
```

The installers copy every folder under `skills/` into:

```text
~/.codex/skills/
```

Restart Codex after installing so it can reload the available skills.

## Notes

This repository should contain skill source files only. Do not commit API keys,
cookies, OAuth files, local caches, or generated bytecode.
